MLinstall | Downloads | Source code | Docs

MLinstall is a free tool that helps install Magic Lantern on Canon cameras. Currently, it can:

  • Make a Canon camera bootable over USB - allowing Magic Lantern to run
  • Show camera info - shutter count, serial number, internal build number
  • Prepare the SD/CF card and make it bootable (Write boot and script flags)
  • Restore the card flags to the default state - without reformatting
